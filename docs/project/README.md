# 2ndEye Project Documentation

This folder is the detailed source of truth for understanding the 2ndEye platform.
If someone wants to understand what the project is, why it matters, how it works, what hardware it supports, and what is required for production readiness, they should start here.

## Scope of This Folder

- Product problem and urgency
- Proposed solution and platform model
- Tech stack and platform strategy
- Users and real-world usage scenarios
- System architecture and module responsibilities
- Hardware ecosystem and integration model
- Reliability and enterprise requirements
- Current Expo MVP baseline and known gaps
- Differentiation, impact, and applicability
- Documentation roadmap for next phases

## Current Product Stage

- Current implementation baseline: Expo app MVP
- Target production implementation: Flutter app with enterprise-grade reliability and scalability
- Product direction: modular, voice-first, background-capable assistive platform with optional connected accessories
