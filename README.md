# 2ndEye

2ndEye is an assistive platform focused on safety, guidance, and accessibility for visually impaired users.
The project combines a mobile app experience, optional hardware integrations, and a FastAPI-first backend direction.

## Current Direction

- FastAPI is the core backend for auth, data, and domain workflows.
- Optional utility services may be used where needed, such as push delivery.
- Flutter is the production mobile target with reliability-first runtime behavior.
